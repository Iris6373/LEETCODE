# Problem

## [55. Jump Game](https://leetcode.com/problems/jump-game/)


# Solution 

## 1) Greedy  - OPTIMAL

      
     
     
   ### Code (.cpp)
   
      class Solution {
      public:
          bool canJump(vector<int>& v) {
              int i, n = v.size(), a = 0;
              a = n-1;
              for (i=n-2;i>=0;i--) {
                  if (i+v[i] >= a) a = i;
              }
              return a == 0;
          }
      };
      
      
     Time Complexity  : O(N) 
                        Since we linearly traverse the array once
     Space Complexity : O(1)
                        Since we no extra space is used
